A Progressive Path

My interest in coding started around the 4th grade. We practiced our math skills using a game site called REFLEX. Eventually, we were given free time on the computers after completing our typing courses. Sometime around then I found out about the Scratch website and decided to recreate one of the REFLEX game.

After that, I was addicted. I made games for everything. If I forgot a birthday gift for my parents I’d quickly make them a game and everything was ok. After a while, I wanted more.

My parents found a local coding club at a nearby library. I enjoyed the club a lot as I got to code and mess around with my friends. There I learned a lot more on Scratch and started to learn C++. Eventually my friends stopped going and I had moved on from Scratch so I also quit.

For the next step, some background is required. I have always enjoyed math and especially large numbers. So, I’ve been slightly addicted to incremental and idle games. Where one the goal of the game is to have a big number keep getting bigger. During quarantine, I found a small modding community around one of the idle games. I joined as I thought it might be interesting. I immensely enjoyed it, but the mod used a framework and made it much easier than base javascript. Which is fine, but I wanted something harder.

So, I started my own game and built it from the ground up. I started to try and soft code the implementation of elements into the game. And I succeeded, but it wasn’t as easy as the mod created for the previous game. Eventually, with a lack of ideas for the game, I decided to stop developing it.

One day, I was messing around in excel trying to quantify love and decided this would be the perfect chance to make another website. And so, I did.

This next one requires a bit of explanation as well. So, I’ve always been a big Minecraft player and I had always enjoyed the technical side of the game beyond the base gameplay. One day, I saw a youtube video of a server that allowed you to code using their custom “language” on the server. I joined and made a lot of projects that never seemed to be completed, but I learned a lot and saw the parallels between javascript and this new “language.” Eventually I wanted to move on to something more challenging. So I started making data packs for the game that were similar to mods, but much harder to create as the language has very little documentation and a small community. It provided a large challenge that I immensely enjoyed and is still the project I’m proudest of.

With the free time I had in math class and a calculator that can program python I decided to try and make some programs to help with school work. I made a couple of math-themed projects that I never used for school. Typing on a calculator is very hard.

Now, I've completed a few projects at NextTech, but this is the one I'm proudest of. It has a framework that makes it very easy to add on branches of the story and comes with the feature that a certain path can require having made certain choses previously.
